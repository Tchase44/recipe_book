# Recipe Book App

## Vue.js Front End

### Project setup and Start with Yarn
```
$ cd vue
$ yarn install
$ yarn run serve
```
### Project setup and Start with NPM
```
$ cd vue
$ npm install
$ npm run serve
```

## Rails API Back End

### Start Server
```
$ cd api
$ bin/rails s
```
